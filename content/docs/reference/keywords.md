This page serves as a quick lookup for all reserved keywords in Sindlish and their English equivalents.

## Core Keywords

| Sindlish | English | Purpose |
| :--- | :--- | :--- |
| **`adad`** | Integer | Whole numbers |
| **`dahai`** | Float | Decimal numbers |
| **`lafz`** | String | Text and characters |
| **`faislo`** | Boolean | `sach` (True) or `koorh` (False) |
| **`khali`** | Null | Represents nothingness |
| **`pakko`** | Constant | Variable that cannot be changed |
| **`kaam`** | Function | Defines a block of code |
| **`wapas`** | Return | Exits a function and returns a value |

## Control Flow

| Sindlish | English | Purpose |
| :--- | :--- | :--- |
| **`agar`** | If | Start of a conditional block |
| **`nahito`** | Else | Alternative block if `agar` fails |
| **`nahita agar`**| Else If | Checks another condition |
| **`tor`** | For | Loop over a range or collection |
| **`jari`** | While | Loop as long as condition is true |
| **`bas`** | Break | Stop the loop immediately |
| **`halo`** | Continue | Skip to the next loop iteration |

## Data Structures

| Sindlish | English | Purpose |
| :--- | :--- | :--- |
| **`fehrist`** | List | Ordered collection |
| **`lughat`** | Dictionary | Key-value pairs |
| **`majmuo`** | Set | Unique collection of items |

## Error Handling

| Sindlish | English | Purpose |
| :--- | :--- | :--- |
| **`ghalti`** | Error | Returns an error state |
| **`bachao`** | Rescue | Provides a default value for a `Result` |
| **`lazmi`** | Mandatory | Forces a value (panics if error) |

## Symbols

| Symbol | Name | Usage |
| :--- | :--- | :--- |
| **`#`** | Hash | Single-line comment |
| **`/* */`** | Slash-Star | Multi-line comment |
| **`?`** | Question | Propagate errors up the stack |
| **`!!`** | Double-Bang | Force unwrap (unsafe) |
| **`->`** | Arrow | Specify function return types |
| **`:`** | Colon | Specify variable types |
